Please use this repo to solidify the GraphDB space we are going to be using.

Also use this repo to build and connect the user interface, and post any other relevant files.

Please download this: https://graphdb.ontotext.com/documentation/11.2/_downloads/9ad0c85c635a3d1db4e23a806691f263/starwars.trig for testing the qwen agent

You can run 'pip install -r requirements.txt' to install all required libraries

IMPORTANT: If running into issues when installing the GPTQModel libraries, try:

'pip install --no-build-isolation GPTQModel==5.7.0'

-> this forces the installation to look for the torch installed in your local .venv instead of systemwide

Setting up agent:

Go to 'console.groq.com' to get setup with your groq API keys
